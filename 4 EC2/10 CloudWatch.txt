CloudWatch:
===========

	* Metrics available by default.

		CPU
		Disk
		Network
		Status

	Monitor EC2 at application Level.


	Dashboards - Create awesome dashboards to see what is happening with your AWS environment.
	Alarms - Allows you to set alarms that notify you when particular thresholds are hit.
	Events - CloudWatch events help you to respond to state changes on your AWS resources.
	Logs - CloudWatch logs helps you to aggregate, monitor and store logs.
